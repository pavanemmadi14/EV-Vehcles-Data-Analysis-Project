# EV Vehicle Analysis Dashboard
## Live Dashboard:
https://public.tableau.com/app/profile/pavan.emmadi/viz/EVVehicleAnalysisDashboard/Dashboard1?publish=yes

## Project Overview:
The EV Vehicle Analysis Dashboard is an interactive data visualization project developed using Tableau to analyze trends in electric vehicles. This dashboard provides insights into EV adoption, vehicle types, manufacturer performance, and electric vehicle growth over time.

Electric vehicles are gaining popularity due to environmental benefits and technological advancements. This project helps analyze the growth of EV adoption and provides meaningful insights using interactive charts and filters.

The dashboard enables users to explore data dynamically and understand patterns in the EV market.

## Project Objectives
-The main objectives of this project are:\
-Analyze electric vehicle trends using data visualization\
-Understand the distribution of different EV types\
-Identify top electric vehicle manufacturers\
-Study EV growth across model years\
-Provide an interactive dashboard for data exploration\

## Dataset Description
The dataset contains information about electric vehicles including manufacturer details, vehicle specifications, and EV adoption statistics.\
Key Attributes\
Vehicle Model\
Manufacturer\
Model Year\
Electric Range\
Vehicle Type (BEV / PHEV)\
State / Region\
Clean Alternative Fuel Eligibility\
These attributes allow analysis of EV adoption trends, manufacturer performance, and technological improvements.

Visualizations Included

## Top EV Manufacturers
Displays manufacturers producing the most electric vehicles.

#  Vehicle Type Distribution
Compares Battery Electric Vehicles (BEV) and Plug-in Hybrid Electric Vehicles (PHEV).

## Geographical Distribution
Shows EV adoption across regions.

## Interactive Filters
Users can filter data based on vehicle type, model year, or manufacturer.

## Key Insights

-Some insights derived from the dashboard include:\
-Electric vehicle adoption has increased significantly in recent years.\
-Battery Electric Vehicles (BEV) dominate the EV market.\
-Certain manufacturers lead the EV industry.\
-EV technology improvements have increased vehicle range.\
-Some regions show higher EV adoption compared to others.
